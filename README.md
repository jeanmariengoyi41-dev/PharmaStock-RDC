# PharmaStock RDC

## Qui je suis
Je suis Jean Marie Ngoyi, étudiant en Génie Civil à l'Université Officielle de Mbujimayi (UOM), Faculté Polytechnique.

## Le projet
PharmaStock RDC est une application Android qui aide les pharmacies de l'Afrique, Kinshasa, mbujimayi et toutes les provinces de la RDC à gérer leurs stocks et à éviter les ruptures de médicaments.

## Problème résolu
Aujourd'hui, une pharmacie à Limete ne sait pas si la pharmacie à Gombe a du Paracétamol. Résultat : les clients font 10 km pour rien.

## Solution
Une app où chaque pharmacie met à jour son stock en 10 secondes. Toutes les autres voient ce qui est disponible en temps réel.

## Tech
- Android natif avec AIDE
- Backend Node.js sur Render
- Base de données PostgreSQL
- Paiement Mobile Money via MaxiCash API
